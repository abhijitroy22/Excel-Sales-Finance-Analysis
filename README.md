# Excel Sales & Finance Analysis

## About the Project

This project analyzes the sales and financial performance of **AtliQ Hardware** using **Advanced Excel**. The solution combines data transformation, data modeling, and interactive reporting to evaluate customer performance, market trends, sales targets, profitability, and Gross Margin across multiple fiscal periods.

---

## Sales Report

### Project Objectives

1. Build a **[Customer Performance Report](./Customer%20Performance%20Report.pdf)**.
2. Conduct a comparative analysis between **[Market Performance and Sales Targets](./Market%20Performance%20vs%20Target.pdf)**.

### Business Value

- Evaluate customer performance across multiple fiscal years.
- Analyze market-wise sales performance and growth trends.
- Compare actual sales against sales targets.
- Support data-driven sales decision-making.

---

## Finance Report

### Project Objectives

1. Build a **[Profit & Loss Report by Fiscal Year](./P%26L%20by%20Fiscal%20Year.pdf)**.
2. Build a **[Profit & Loss Report by Month](./P%26L%20by%20Month.pdf)**.
3. Build a **[Profit & Loss Report by Market](./P%26L%20by%20Markets.pdf)**.

### Business Value

- Analyze revenue, expenses, and profitability over different time periods.
- Compare financial performance across markets.
- Evaluate Gross Margin trends.
- Support business planning through financial reporting.

## Project Highlights

- Performed ETL by cleaning and transforming raw datasets using Power Query.
- Integrated multiple datasets into a unified data model using Power Pivot.
- Created DAX measures and calculated columns for business reporting.
- Developed interactive Sales Analysis and Finance (P&L) reports.
- Analyzed customer, product, market, and regional performance.
- Generated fiscal year, quarterly, and monthly business reports.
